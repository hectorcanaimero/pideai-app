# PideAI Admin — App Store Listing

## App Name
PideAI Admin

## Subtitle (iOS) / Short Description (Android)
Gestiona tu tienda desde cualquier lugar

## Description (Both Platforms)

PideAI Admin es la app para administrar tu tienda o restaurante desde el celular. Recibí pedidos en tiempo real, gestioná tu menú, controlá tu negocio y mantené a tus clientes contentos — todo desde tu bolsillo.

**Funcionalidades principales:**

📊 Dashboard en tiempo real
- Estadísticas del día: pedidos, ventas, ticket promedio
- Control de estado de la tienda (abrir/cerrar)
- Alertas de stock bajo

📋 Gestión de pedidos
- Recibí pedidos con notificaciones push
- Cambiá el estado con un toque
- Kitchen Display System para la cocina
- Historial completo de pedidos

🍽️ Menú y productos
- Categorías y productos con fotos
- Control de stock y disponibilidad
- Extras y grupos de extras
- Subí fotos desde la cámara o galería

👥 Clientes
- Directorio de clientes
- Historial de compras por cliente
- Contacto directo: llamar, WhatsApp, email

⚙️ Configuración completa
- Horarios de atención
- Métodos de pago
- Conversión de moneda
- Zonas de delivery
- Diseño de la tienda

📈 Analíticas
- Ventas por período
- Productos más vendidos
- Métricas de clientes

🔔 Notificaciones
- Push notifications en nuevos pedidos
- Badge count de pedidos pendientes

**Requisitos:**
- Cuenta activa en PideAI (pideai.com)
- Conexión a internet
